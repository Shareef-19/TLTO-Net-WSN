
# TLTO-Net — Programmatic Figure Generation

This repository contains Python code used to generate simulation figures for:

**"Adaptive Trust and Energy Optimization in Wireless Sensor Networks (TLTO-Net Model)"**

## Contents
```
/src/TLTO_Net_Code.py   → Full executable Python code
/figures/               → You may save output graphs here
```

## How to Run
1. Install dependencies:
```
pip install numpy matplotlib
```

2. Run the script:
```
python TLTO_Net_Code.py
```
